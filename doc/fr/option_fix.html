Option : fix

Syntaxe

-fix <axe> <above|below> <d> <normale>

Description

Cette option permet de figer la position de certains atomes suivant X, Y et/ou Z.

Les paramètres associés à cette option sont :

  • axe : direction suivant laquelle les atomes seront figés, peut être 'x', 'y', 'z' (axes cartésiens), ou 'all' (ce dernier indiquant que les atomes seront figés dans les trois directions de l'espace).
  • above|below : ce mot-clé décide si ce sont les atomes au-dessus (above) ou en-dessous (below) du plan qui seront figés.
  • d : distance entre le plan et l'origine, en angströms (Å).
  • normale : normale au plan définissant la sélection ; peut être 'x', 'y', 'z', ou une direction cristallographique (voir ci-dessous).

La distance <d> est généralement donnée en Å. Il est également possible de spécifier une distance relative aux dimensions de la boîte avec le mot-clé BOX et une opération (voir cette page).

La direction <normale> au plan peut être 'x', 'y', 'z' (axes cartésiens), ou une direction cristallographique définie par ses indices de Miller (voir comment spécifier les indices de Miller). Si l'orientation crystallograpĥique du système est définie (par exemple grâce à l'option -properties) alors les indices de Miller se réfèrent à cette orientation. Sinon il est supposé que l'orientation est X=[100], Y=[010], Z=[001].

Notez que cette option ne rend pas ces atomes inertes pour atomsk, elle ne fait que définir l'état (figé ou non) des atomes comme propriétés auxiliaires. Les options de atomsk ne traitent pas les atomes figés différemment des autres atomes. Ce n'est que lors de l'écriture d'un fichier de sortie que ces atomes seront marqués différemment si le format le permet (voir la liste des formats). Pour restreindre les options de atomsk à certains atomes il est possible d'utiliser l'option -select.

Cette option peut être utilisée plusieurs fois pour figer les atomes dans différentes directions, ou pour figer des atomes différents (voir l'exemple 2 ci-dessous).

Si certains atomes ont des coquilles (dans le cadre d'un modèle ionique cœur-coquille), seules les positions des cœur sont figées (mais pas les positions des coquilles).

Si une sélection est définie (avec l'option -select) alors seuls les atomes sélectionnés seront fixés.

Défaut

Par défaut aucun atome n'est figé.

Exemples

  • atomsk initial.xsf -fix y final.gin

    Ceci lira le fichier initial.xsf, et figera la composante Y de tous les atomes. Le résultat sera écrit dans le fichier final.gin.

  • atomsk system.xsf -fix x above 5 z gin

    Ceci lira system.xsf, et figera la composante X de tous les atomes situés au-dessus de 5 Å suivant Z. Le résultat sera écrit dans system.gin.

  • atomsk initial.xsf -fix x below 2.4 z -fix y below 2.4 z final.gin

    Ceci fixera les composantes X et Y des atomes situés en-dessous de 2.4 Å suivant Z.

  • atomsk initial.xsf -fix all below 0.2*BOX z -fix all above 0.8*BOX z final.gin

    Ceci figera complètement les atomes situés en-dessous de 0.2 fois la hauteur de boîte suivant Z, ou au-dessus de 0.8 fois cette hauteur.

  • atomsk initial.cfg -select out box 10 10 0 60 60 100 -fix all final.bop

    Ceci va sélectionner les atomes dans une région, et figera ces atomes dans les trois directions de l'espace. Ceci peut être utile pour figer les atomes en bord de boîte.
