Option : remove-atom

Syntaxe

-remove-atom <indice>

-rmatom <indice>

-rmatom <espèce>

Description

Cette option supprime l'atome ayant l'indice donné, ou tous les atomes de l'espèce donnée.

Si le ou les atomes supprimés ont une coquille (dans le cadre d'un modèle cœur-coquille) et/ou des propriétés auxiliaires, ils sont également supprimés. Par ailleurs, les atomes restant dans le système peuvent avoir un indice différent après l'application de cette option.

Si une sélection est définie (avec l'option -select) alors seuls les atomes sélectionnés seront supprimés.

Défaut

Par défaut aucun atome n'est supprimé.

Exemples

  • atomsk initial.cfg -rmatom 1622 final.xsf

    Ceci lira le fichier initial.cfg, supprimera l'atome #1622, et écrira le résultat final.xsf.

  • atomsk initial.cfg -rmatom Fe final.xyz

    Ceci supprimera tous les atomes de fer du système.

  • atomsk initial.cfg -select in sphere 0.5*box 0.5*box 0.5*box 30 -rmatom Si final.xyz

    Ceci va sélectionner les atomes dans une sphère, et supprimera les atomes de silicium qui font partie de cette sélection.
