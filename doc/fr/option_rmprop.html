Option : remove-property

Syntaxe

-remove-property <propriété>

-rmprop <propriété>

Description

Cette option supprime une ou plusieurs propriétés auxiliaires associées aux atomes.

La <propriété> fournie doit correspondre à une propriété existante, lue depuis le fichier d'entrée ou grâce à l'option -properties. Si <propriété> vaut "all" alors toutes les propriétés auxiliaires sont supprimées.

Si une sélection est définie (avec l'option -select) alors la propriété ne sera pas supprimée. Au lieu de cela une valeur de zéro sera donnée à la ou les propriétés.

Défaut

Par défaut aucune propriété auxiliaire n'est supprimée, mais elles ne sont écrites dans les fichiers de sortie que si leur format le supporte.

Exemples

  • atomsk systeme.cfg -rmprop all xsf

    Ceci lira le fichier systeme.cfg et le convertira vers system.xsf, supprimant toutes les propriétés auxiliaires.

  • atomsk systeme.lmc -rmprop fx -rmprop fy -rmprop fz exyz

    Ceci lira le fichier system.lmc et supprimera les trois composantes des forces fx,fy,fz.
