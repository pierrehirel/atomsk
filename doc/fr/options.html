Options

Les options sont des opérations qui transforment la configuration atomique, par ex. une rotation, ajouter, supprimer ou déplacer des atomes, etc.

Plusieurs options peuvent être appelées dans une même ligne de commande. Elles seront alors appliquées consécutivement, dans l'ordre où elles sont appelées. Les options peuvent apparaître n'importe où dans la ligne de commande (par ex. avant ou après le fichier d'entrée, avant ou après les formats de sortie), cependant lorsqu'une option est invoquée elle doit être suivie de ses paramètres dans le bon ordre.

Les options s'appliquent à tous les fichiers traités lors d'une exécution du programme. Par exemple, si le mode 'list' est employé, alors les options s'appliqueront à tous les fichiers de la liste. Veuillez vous référer à la page décrivant chaque mode pour savoir comment ils interagissent avec les options.

L'option -select est un cas spécial, puisqu'elle peut être utilisée pour restreindre les options suivantes à certains atomes. Veuillez vous référer à la page correspondante pour en savoir plus.

Dans les options utilisant des directions cristallographiques, les indices de Miller peuvent être soit contigus, soit séparés par un tiret bas. L'utilisation de crochets est optionelle. Par exemple la direction [110] peut être spécifiée par "1-10", "[1-10]", "1_-1_0" ou "[1_-1_0]" (sans guillemets). Notez qu'en aucun cas les indices d'une direction ne doivent être séparés par des espaces.

Dans certaines options requérant une position, il est possible de fournir une position cartésienne en angströms, ou une position relative grâce au mot-clé. Consultez cette page pour en savoir plus.

Ci-dessous les options sont listées par ordre alphabétique.

Liste des options

  • add-atom - Insérer un ou plusieurs nouveaux atomes
  • add-shells - Créer des coquilles pour certains atomes
  • alignx - Aligner le premier vecteur de boîte avec l'axe X
  • bind-shells - Re-associer les nuages électroniques avec leur cœur
  • center - Translater le système au centre de la boîte
  • crack - Insérer une fissure dans le système
  • cut - Supprimer une partie du système
  • deform - Appliquer une contrainte ou une déformation uniaxiale
  • dislocation - Insérer une dislocation dans le système
  • disturb - Perturber les positions des atomes aléatoirement
  • expand - Dupliquer le système, construire des super-cellules
  • fix - Fixer les coordonnées de certains atomes
  • fractional - Convertir en coordonnées réduites
  • mirror - Appliquer une transformation miroir
  • orient - Changer l'orientation crystallographique
  • properties - Définir des propriétés du système
  • rebox - Tente de déterminer les vecteur de boîte
  • remove-atom - Supprimer un ou plusieurs atomes
  • remove-doubles - Supprimer les atomes en double
  • remove-property - Supprimer une ou plusieurs propriétés auxiliaires
  • remove-shells - Supprimer les coquilles sur certains ions
  • rotate - Tourner le système autour d'un axe
  • select - Sélectionner les atomes à l'intérieur ou à l'extérieur d'une région
  • shear - Appliquer un cisaillement
  • shift - Translater une partie du système
  • sort - Changer l'ordre dans lequel les atomes sont rangés
  • substitute - Remplacer des atomes par d'autres atomes
  • unit - Changer de système d'unités
  • unskew - Réduire le cisaillement de la boîte
  • velocity - Générer une distribution de vitesses de Maxwell-Boltzmann
  • wrap - Remballer les atomes dans la boîte
