Option : torsion

Syntaxe

-torsion <axe> <θ>

Description

Cette option permet d'appliquer une torsion le long d'un axe cartésien.

Il est nécessaire de fournir l'axe de rotation (x, y ou z), et l'angle de rotation θ (en degrés).

L'axe de rotation est placé au centre du plan normal à l'<axe>. Pour effectuer une rotation autour d'un autre point il est possible d'employer l'option -shift.

Si les forces (fx,fy,fz) ou les vitesses (vx,vy,vz) existent comme propriétés auxiliaires, alors la rotation s'appliquera aussi à ces vecteurs.

Si une sélection est définie (avec l'option -select) alors la rotation ne sera appliquée qu'aux atomes sélectionnés, et la boîte ne sera pas modifiée. Certains atomes peuvent alors se retrouver hors de la boîte, ce qui peut être corrigé grâce à l'option -wrap.

Cette option n'affecte pas les vecteurs de boîte, ni le tenseur élastique s'il est défini.

Défaut

Par défaut aucune torsion n'est appliquée.

Examples

  • atomsk initial.cfg -torsion x 10 final.xyz

    Ceci appliquera une torsion de 10 degrés suivant X.
