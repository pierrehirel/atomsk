Comportement du programme : fichier journal

Syntaxe

-log <fichierjournal>

Description

Cette option permet de changer le nom du fichier dans lequel atomsk écrit son journal de progression ("fichier log"). Par défaut ce fichier est atomsk.log.

Notez que si cette option est utilisée alors le niveau de verbosité sera automatiquement égal à 3, sauf si un niveau de verbosité différent de 0 ou 1 est spécifié.

Pour rendre ce changement permanent, vous pouvez éditer un fichier de configuration.

Défaut

Par défaut le fichier journal est atomsk.log.

Exemples

  • atomsk initial.gin cfg -log myrun.log

    Ceci écrira la progression du programme sur lécran et dans le fichier myrun.log.

  • atomsk initial.gin cfg -log myrun.log -v 0

    Identique à l'exemple précédent. Puisqu'un nom de fichier journal est spécifié, le niveau de verbosité ne peut pas être égal à zéro -il est réinitialisé à 3.

  • atomsk initial.gin cfg -log myrun.log -v 2

    Ceci écrira la progression du programme seulement dans le fichier myrun.log (aucun message ne sera affiché à l'écran, sauf les messages d'erreur).
