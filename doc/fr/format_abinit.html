Format : ABINIT

Nom : fichier d'entrée ABINIT

Extension : in

Spécification : documentation d'ABINIT

Programmes de visualisation : N/A

Restrictions

Atomsk peut lire les fichiers d'entrée d'ABINIT (*.in). Si les coordonnées sont réduites (avec le mot-clé xred) alors Atomsk les convertit immédiatement en coordonnées cartésiennes. Pour les convertir de nouveau en coordonnées réduites, il est possible d'utiliser l'option -frac.

Lorsqu'il écrit un fichier au format ABINIT, Atomsk écrit les positions atomiques en coordonnées cartésiennes (avec le mot-clé xcart). Si le "type" des atomes est défini comme propriété auxiliaire, alors Atomsk l'utilise. Le fichier produit est juste un brouillon et devra être édité, il n'est PAS suffisant pour une simulation.

Exemples

  • atomsk my_system.cfg abinit

    Ceci va lire le fichier my_system.cfg, et écrira le résultat final dans my_system.in.
