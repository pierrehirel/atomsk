Aperçu

But

Atomsk est un programme en ligne de commandes conçu pour créer, manipuler, et convertir des systèmes atomiques. Il supporte de nombreux formats de fichiers, dont ceux de LAMMPS, VASP, Quantum Espresso, Atomeye, et xCrySDen (voir la liste des formats supportés), ce qui facilite la conversion de fichiers pour les calculs ab initio, les simulations avec des potentiels inter-atomiques, ou la visualisation. De plus atomsk peut aussi effectuer des transformations simples sur les positions atomiques (grâce aux nombreuses options), comme créer des super-cellules, couper des plans cristallographiques, appliquer une contrainte, ou encore insérer des dislocations.

Notez cependant que ce programme n'a pas pour vocation de préparer tous les fichiers, ni définir tous les paramètres de simulation, pour quelque utilisation particulière que ce soit. Dans la plupart des cas atomsk produira un fichier de positions atomiques qui ne sera qu'un brouillon, et qui devra être édité, modifié et/ou complété avant d'effectuer une simulation.

Démarrage rapide

La façon la plus simple d'obtenir ce code est de télécharger l'exécutable pour votre plate-forme depuis cette page. Si vous préférez compiler le code vous-même, lisez ces instructions détaillées.

Ce programme est conçu pour être simple à utiliser. Si vous n'y connaissez rien en programmation, ne savez pas par où commencer, ou voulez simplement convetir un fichier, exécutez simplement le programme en ligne de commande :

atomsk

et suivez les instructions. Une autre façon d'utiliser le programme pour convertir un fichier est d'utiliser le mode normal, qui permet de convertir un fichier vers un format donné :

atomsk <fichier> <format>

Des exemples simples sont illustrés sur cette page.

Obtenir de l'aide en ligne de commandes

Pour afficher un bref rappel des options en ligne de commande, tapez :

atomsk --help

Utilisation avancée

Malgré cette volonté de simplicité, atomsk permet également de réaliser des choses plus élaborées, comme :

  • conversion simultanée vers plusieurs formats ;
  • convertir une liste de fichiers ;
  • joindre deux ou plusieurs systèmes ;
  • créer des systèmes simples à partir de rien ;
  • créer un polycrystal ;
  • calculer les différences de positions atomiques entre deux systèmes ;
  • appliquer des transformations sur les positions atomiques ;
  • des paramètres personnels peuvent être enregistrés dans un fichier de configuration.

Pour une utilisation avancée, référez-vous aux sections de l'aide en ligne.
