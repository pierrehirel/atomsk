Option : wrap

Syntaxe

-wrap

Description

Cette option replace les atomes dans la boîte. Autrement dit, tous les atomes qui se trouveraient à l'extérieur de la boîte (définie par les vecteurs de la boîte H) sont translatés d'un ou plusieurs vecteurs de boîte jusqu'à ce qu'ils se trouvent dans la boîte.

Si certains atomes ont des coquilles (dans le sens d'un modèle ionique cœur-coquille) alors les coquilles seront déplacées du même vecteur que leur cœur, de sorte que le vecteur entre cœur et coquille reste le même. Cela signifie que même si tous les cœurs sont bien replacés à l'intérieur de la boîte, certaines coquilles peuvent se retrouver à l'extérieur de la boîte.

Si une sélection est définie (avec l'option -select) alors seuls les atomes sélectionnés seront replacés dans la boîte.

Défaut

Par défaut les atomes ne sont pas replacés dans la boîte.

Exemples

  • atomsk initial.xsf -wrap final.xyz

    Ceci lira le fichier initial.xsf et remballera tous les atomes dans la boîte. Le résultat sera écrit dans final.xyz.
