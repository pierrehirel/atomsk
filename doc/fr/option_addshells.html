Option : add-shells

Syntaxe

-add-shells <espèce>

-as <espèce>

Description

Cette option crée des coquilles (dans le cadre d'un modèle ionique cœur-coquille) pour tous les atomes d'une espèce chimique donnée. Si <espèce> vaut "all" alors des coquilles seront créées pour tous les atomes.

Si certains atomes possédaient déjà une coquille et que <espèces> vaut "all", alors les coquilles existantes seront d'abord supprimées, puis de nouvelles seront créées pour tous les atomes.

Notez que les positions des coquilles ne peuvent être écrites que vers certains formats de fichiers, comme le format GIN ou le format de DL_POLY. Lors de l'écriture d'un tel système vers des formats ne le supportant pas, les positions des coquilles seront perdues.

Défaut

Par défaut aucune coquille n'est introduit.

Exemples

  • atomsk initial.cfg -add-shells O final.gin

    Ceci créera des coquilles pour les atomes d'oxygène.

  • atomsk initial.cfg -as all final.gin

    Ceci créera des coquilles pour tous les atomes.

  • atomsk initial.cfg -as Ti -as O final.gin

    Ceci créera des coquilles pour les atomes de titane et d'oxygène.
