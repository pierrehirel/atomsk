Mode : average

Syntax

atomsk --average <fichierliste> <fichiersortie> [<formats>] [options]

Description

Dans ce mode, Atomsk moyenne les positions atomiques d'une liste de fichiers. Ceci peut être utile par ex. pour obtenir une moyenne temporelle des positions atomiques à partir de configurations issues d'une simulation en dynamique moléculaire à haute température.

Le fichier <fichierliste> est un fichier texte qui doit contenir une liste de fichiers à traiter (un nom de fichier par ligne). Les lignes vides et celles commençant par # sont ignorées (ceci peut être utilisé pour insérer des commentaires). Tous les fichiers doivent contenir exactement le même nombre d'atomes, et correspondre au même système (c-à-d. mêmes vecteurs de boîte, mêmes types d'atomes, etc.) pour que le calcul ait du sens. Les fichiers peuvent être dans des formats différents, tant que ces formats sont pris en charge par Atomsk.

Les dimensions de la boîte seront aussi moyennées. Si elles sont les mêmes pour tous les fichiers alors elles seront aussi identiques dans le fichier de sortie. Si elles changent dans certains fichiers, alors le fichier final contiendra une moyenne des différentes boîtes et des coordonnées réduites des atomes.

Si tous les fichiers contiennent des coquilles (dans le cadre d'un modèle ionique cœur-coquille) alors les positions des coquilles seront aussi moyennées. Si tous les fichiers contiennent les mêmes propriétés auxiliaires alors elles seront aussi moyennées, à l'exception des types d'atomes. Si un ou plusieurs fichiers ne contiennent pas de coquilles (ou de propriétés auxiliaires) alors les coquilles (respectivement les propriétés auxiliaires) seront ignorées et seront absentes du fichier final.

Si ce mode est utilisé avec des options alors ces options seront appliquées au fichier final (celui contenant les positions moyennées).

Exemples

  • atomsk --average liste.lst moyenne.cfg

    Ceci va lire tous les fichiers dont les noms sont donnés dans liste.lst, et produira une moyenne des positions atomiques de tous ces systèmes. Le résultat final sera écrit dans moyenne.cfg.

  • ls *.xsf > liste.lst && atomsk --average liste.lst -duplicate 2 2 2 moyenne.cfg xsf

    Ceci va lister tous les fichiers portant l'extension .xsf et enregistrera la liste dans liste.lst, puis exécutera Atomsk pour produire une moyenne des positions atomiques de tous ces systèmes. Le système moyenné sera dupliqué deux fois dans chaque dimension (grâce à l'option -duplicate), et enfin le résultat final sera écrit dans moyenne.cfg et moyenne.xsf.
