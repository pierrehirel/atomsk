Program behaviour

By default the program Atomsk tries to be polite: it asks if files should be overwritten, explains what it does on the screen, trying not to overflow the user with useless information. This default behaviour can be changed with the command-line parameters below, or by setting up a configuration file in the user's home directory.

These parameters will apply all along a run if they appear anywhere in the command-line.

  • ignore - Always ignore existing files
  • language - Set language of messages
  • logfile - Set up log file name
  • nthreads - Set number of OpenMP threads
  • overwrite - Always overwrite existing files
  • verbosity - Set verbosity level
