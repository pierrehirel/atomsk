Format: XV

Name: SIESTA XV format

Extension: XV

Specification: SIESTA documentation

Visualization programs: Aten

Restrictions

atomsk can read and write files in the SIESTA XV format. Note that this format can contain atom positions in Bohrs, however atomsk does not automatically convert them. If needed, one can use the option -unit to perform the conversion before writing or after reading an XV file.

The velocities of atoms are read from and written to XV files. If velocities are undefined then zeros are written instead.

Examples

  • atomsk my_system.cfg xv

    This will read the file my_system.cfg, and convert it to my_system.xv.
