Format: ABINIT

Name: ABINIT input file

Extension: in

Specification: ABINIT documentation

Visualization programs: N/A

Restrictions

Atomsk can read ABINIT input files (*.in). If coordinates are reduced (i.e. if they appear after xred) then Atomsk will immediately convert them to cartesian coordinates. To convert back to reduced coordinates one can use the option -frac.

When writing a file in the ABINIT input format, Atomsk writes atom positions as Cartesian coordinates (using the keyword xcart). If the atom types are defined as an auxiliary properties, then Atomsk will use it. That file is just a draft and will have to be edited, it is NOT fully set for a simulation.

Examples

  • atomsk my_system.cfg abinit

    This will read the file my_system.cfg, and write the final result to my_system.in.
