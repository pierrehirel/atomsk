Format: dd

Name: ddplot file

Extension: dd

Specification: DDplot documentation

Visualization programs: ddplot

Restrictions

Output to ddplot format is available only through the ddplot mode. Refer to the description of this mode for more details.

Examples

  • atomsk --ddplot perfect.xsf dislo.cfg my_system.dd

    This will read the file perfect.xsf containing the configuration with no dislocation, and the file dislo.cfg containing the system with a dislocation, and write the ddplot file my_system.dd.
