Program behaviour: language

Syntax

-language <language>

-lang <language>

Description

This option changes the language of messages produced by Atomsk.

The languages available for now are:

  • de for German.
  • en for English.
  • fr for French.

Default

By default, on GNU/Linux systems the environment variable $LANG is read, and if it is set to a language in the list above then Atomsk will use that language. On Microsoft® Windows® environments English is used by default. This default is overridden by the one defined in the user configuration file. Finally, the command-line option -lang overrides all previous definitions.

Examples

  • atomsk initial.gin cfg -lang fr

    This will read initial.gin and convert it to CFG. All messages will be displayed in French.
